Funcionalidades principais

Cadastro de receitas e despesas

Registre seus ganhos e gastos por categoria, conta e data para ter uma visão clara do seu fluxo de caixa.

Painel com gráficos

Visualize seus gastos por categoria com gráficos que ajudam a entender para onde está indo o seu dinheiro.

Histórico de transações

Consulte o histórico completo de receitas e despesas para acompanhar sua evolução ao longo do tempo.

Metas de economia

Defina objetivos financeiros e acompanhe quanto ainda falta para atingi-los.

Como funciona a arquitetura

1

Esta página é um front-end estático hospedado no GitHub Pages, dentro da pasta docs/ do repositório.

2

Ao clicar em Entrar ou Criar conta, você é redirecionado para o backend Flask hospedado no Render.

3

Todo o processamento (login, cadastro, dashboard, gráficos, etc.) continua sendo feito normalmente pelo backend.

Sobre o projeto

Este projeto faz parte do PI e foi desenvolvido utilizando Flask no backend, com templates, gráficos e gerenciamento de sessões para controle financeiro pessoal.

A hospedagem está separada em duas camadas: o frontend estático no GitHub Pages e o backend no Render.
