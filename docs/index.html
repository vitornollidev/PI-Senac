Sobre o Projeto

FinGest é uma aplicação web desenvolvida para ajudar usuários a organizarem suas finanças pessoais. O sistema permite registrar movimentações financeiras, gerenciar contas, definir metas de poupança e acompanhar relatórios detalhados através de um dashboard interativo.

Funcionalidades

Autenticação de Usuários

Sistema completo de registro e login seguro.

Gestão de Contas

Criação e gerenciamento de múltiplas contas bancárias.

Controle de Transações

Registro de receitas e despesas com categorização.

Dashboard Financeiro

Visualização de gráficos e resumo financeiro.

Metas de Poupança

Definição e acompanhamento de objetivos financeiros.

Histórico de Transações

Consulta detalhada das movimentações financeiras.

Tecnologias Utilizadas

Python & Flask

Backend da aplicação responsável pela lógica de negócio e rotas.

SQLite & SQLAlchemy

Banco de dados utilizado para persistência das informações.

HTML5, CSS3 e JavaScript

Frontend responsável pela interface e experiência do usuário.

Werkzeug

Utilizado para criptografia e segurança das senhas.

pytz

Gerenciamento de fusos horários no sistema.
